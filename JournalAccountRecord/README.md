# JournalAccountRecord
流水账记录软件  
* Blog：[http://techieliang.com](http://techieliang.com/)    
有Qt制作，包含流水类目划分及流水审核   
采用Sqlite作为数据库，若需要其他数据库需在Qt安装对应驱动
